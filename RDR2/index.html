O BresoRDR2 Menu é uma modificação para o Red Dead Redemption 2, onde o jogador tem acesso a varias funcionalidades, desde ficar imortal, ate criar veiculos e NPCs do jogo, de forma muito facil. Com ele, voce tera uma nova experiencia de jogo, podendo explorar locais onde o jogador nao tem acesso, por exemplo, indo ate a ilha de Guarma, que só é acessivel em missao, teleportar para varios locais, usar veiculos especiais, gerar guardacostas para acompanhar voce pelo mapa, explorar o fundo do oceano, entre centenas de outras funcoes que o mod proporciona. Lembrando que o mod serve apenas para o modo OFFLINE do jogo, na versao 1.0 do jogo (versao inicial, sem DLCS) e que nao pode ser acessado no modo online para subir as estatisticas do personagem ou burlar o sistema do jogo. Use apenas para ter uma nova visao do jogo e tudo oque pode ser feito com ele
